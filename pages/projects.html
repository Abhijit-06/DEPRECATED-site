Our Projects

Software Carpentry is an open project. Like all such projects, it needs your help to grow:

  • Our core lessons have been extracted from this repository and put into repositories of their own—please see the lessons page for links. We hope to finish this work by the end of January 2015; issues and pull requests to help us get there would be very welcome.

  • You can also help improve the templates for our workshop websites and new lessons. We would be particularly grateful for help building a validation tool for lessons to help check that everything is in the right format—please see the issues and pull requests for details.

  • We have started building a web-based tool for managing workshops, which we hope will relieve our biggest bottleneck. If you can program in Python and are familiar with Django, we would be very grateful for your help.

  • ipythonblocks and skimage.novice are allowing us adopt a media-first approach to teaching programming in Python. Enhancements to either of these are very welcome.

  • Several of our team are experimenting with a pure HTML replacement for screencasts called Browsercast. We plan to use it in our slideshows, but we think it has much wider application.

  • A groups of students have built a diff-and-merge tool for the IPython Notebook. Polishing this would help everyone in the IPython community, not just us.
