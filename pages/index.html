Our mission is to teach researchers basic lab skills for scientific computing: the tools and techniques that will help them get more done in less time, and with less pain. Our core two-day curriculum includes:

  • the Unix shell (and how to automate repetitive tasks);
  • Python or R (and how to grow a program in a modular, testable way);
  • Git and GitHub (and how to track and share work efficiently); and
  • SQL (and the difference between structured and unstructured data).

Our volunteer instructors have run over a hundred events for several thousand scientists in the past two and a half years. All of our lesson materials are freely reusable under the Creative Commons - Attribution license,
