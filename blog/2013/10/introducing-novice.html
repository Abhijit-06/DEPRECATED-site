Intro

Opening and displaying an image is easy. Just import the novice modul and call open. The picture will display inside an IPython notebook.

from skimage import novice
mushroom_url = "http://i.imgur.com/WNNVM4X.png"
picture = novice.open(mushroom_url)
picture

Image manipulation is limited, but very simple. You can grab chunks of the image using slicing notation. Note that skimage.novice uses Cartesian coordinates, so (0, 0) is the bottom left.

picture[100:150, :200]  # Left (100) to right (150), bottom (0) to top (200)

Slices can even contain a step size, so flipping an image upside down is a piece of cake!

picture[:, ::-1]  # All horizontal, all vertical backwards

Pictures can be iterated over with the standard for syntax, and the resulting pixel objects refer back to the original picture. Below, we search for pixels whose green components are more than half full (255 is the max), and swap their red and green values. The result is a red mushroom!

for pixel in picture:
    if pixel.green > 128:  # Red, green, and blue go from 0 to 255
        tmp = pixel.red  # Swap red and green
        pixel.red = pixel.green
        pixel.green = tmp

To resize an image, just assign a new (width, height) tuple to the size property of the picture. The novice.Picture class has a static from_size method to create empty images. The example below shows how to create a collage of differently-colored mushrooms.

red_mushroom = picture
red_mushroom.size = (50, 50)  # Resize to 50x50

# Grab another copy of the original image
green_mushroom = novice.open(mushroom_url)
green_mushroom.size = (50, 50)  # Resize to 50x50

# Create a blank 100x100 image filled with black.
# See novice.colors for more colors.
garden = novice.Picture.from_size((100, 100), color="black")
garden[:50, :50] = red_mushroom    # bottom left
garden[50:, :50] = green_mushroom  # bottom right
garden[:50, 50:] = green_mushroom  # top left
garden[50:, 50:] = red_mushroom    # top right
garden
