The Mozilla Science Lab is helping a global network of researchers, tool developers, librarians and publishers collaborate to further science on the web. They do this through projects like Collaborate which brings developers and researchers together around selected projects, prototype efforts such as Code as a Research Object, and develop curriculum with the community on topics like Code Review and stand-alone lessons on topics like user testing in the sciences.
