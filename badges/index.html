The web and other new learning spaces provide many ways to gain skills and experience—from online courses, learning networks, mentorship, and peer learning. Badges provide an open, portable, verifiable way for learners to get recognition for these skills. Using Mozilla's Open Badge Infrastructure, Software Carpentry is issuing three kinds of badges:

  • Creator,
  • Instructor,
  • Organizer.

Our issuer information can be found here.
